## 본격 필수 Git 명령어!
`git diff`  
버그같은 현상이 있어서 잘 안 쓴다. 그래서 대신 모 쓴다?
**git difftool!**  
Vim 에디터로 시각적으로 잘 달라진 점을 보여준다. 내가 진짜 개발을 하고 있다는 기분은 덤!

`git difftool` + commit id  
뒤에 커밋 아이디를 적으면 직접 비교해볼 수 있다  
`git log --oneline --all`  
이 명령어로 치면 커밋 내역과 아이디를 확인할 수 있다
